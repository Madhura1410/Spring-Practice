# Spring-Practice
🚀 My Spring Framework practice repository showcasing daily coding exercises, projects, and concepts from beginner to advanced.

📖 About

This repository documents my hands-on journey learning the Spring Framework. Each folder captures a specific concept or exercise, progressing from core Dependency Injection basics toward more advanced Spring topics — built for consistent, daily practice and as a portfolio of my backend development skills.

🧠 Topics Covered

Dependency Injection — Setter Injection
Dependency Injection — Setter Injection with Primitives
Dependency Injection — Constructor Injection
Dependency Injection — Constructor Injection with Primitives
Spring Beans & the IoC Container
Core Spring configuration concepts



This list grows as new exercises are added — see the folder structure below for the current state.
